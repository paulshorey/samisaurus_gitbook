# 3/4 letter "root", add suffix

### if ends in one consonant:

```text
root(ia)  
root(io)  
root(ico)  
```

### if end in "d":

```text
food(lio)  
vid(lio)  
med(lio)  
```

### if ends in two consonants:

```text
camp(er)  
camp(us)  
camp(ix)  
camp(or)  
camp(ar)  
```

### if ends in one vowel:

```text
taco(xin)  
casa(xin)  
```

### if ends in two vowels:

```text
asia(tic)  
asia(tion)  
ego(t)(ist)  
```
