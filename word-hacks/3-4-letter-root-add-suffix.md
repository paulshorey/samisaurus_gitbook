# 3/4 letter "root", add suffix

### if ends in one consonant:

```text
root(ia)  
root(io)  
root(ico)  
```

### if ends in "l", "le", "ly":

```text
meddl(io)  
curl(io)  
```

### if end in "d":

```text
food(lio)  
vid(lio)  
med(lio)  
```

### if ends in two consonants:

```text
camp(er)  
camp(us)  
camp(ix)  
camp(or)  
camp(ar)  
```

### if ends in "x":

```text
pix(ico)  
box(ar)  
fox(us)  
```

### if ends in one vowel:

```text
taco(xin)  
casa(xin)  
```

### if ends in two vowels:

```text
asia(tic)  
asia(tion)  
ego(t)(ist)  
```

### if 3rd/4th letter is "t"

```text
pet(us)  
hort"iculture" - hort(us)  
dict"ionary" - dict(us)  
rent(us)  
```
